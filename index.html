Asset Allocation

This program finds a list of the most opimized stock portfolios given a list of assets using the Sharpe ratio and the Monte Carlo method. I then plot the past performance of each generated portfolio and a scatterplot of the return and risk of each portfolio. This scatterplot visualizes the concept of the efficient frontier.
